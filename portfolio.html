  • Descripcion: Desarrollo de un bot de Telegram para automatizar tareas administrativas dentro de la institución de salud, como la gestión de guardias y el procesamiento de solicitudes de licencias.

  • Herramientas Utilizadas: Node.js, Base de Datos Relacional.

  • Rol: Desarrollador.

  • Resultados: La automatización mejoró la eficiencia administrativa y la precisión en el manejo de datos.

Implementación Aplicación Web de Mecánica Pulmonar en la Historia Clínica Electrónica - Instituto Argentino de Diagnóstico y Tratamiento (2021-Actualidad)

  • Descripción: Desarrollo e implementación de una aplicación web para calcular la mecánica pulmonar integrada con la historia clínica digital del sanatorio.

  • Herramientas Utilizadas: JavaScript, HTML, CSS (Frontend), Node.js (Backend).

  • Rol: Desarrollador.

  • Resultados: La herramienta permitió la visualización de datos en tiempo real, facilitando la toma de decisiones clínicas.

Gestión de Datos en REDCap - Clínica Basilea (2022-2024)

  • Descripción: Implementación de procesos de recolección y gestión de datos utilizando REDCap, asegurando la calidad y coherencia de la información almacenada.

  • Herramientas Utilizadas: REDCap, Base de Datos Relacional.

  • Rol: Líder de Proyecto.
